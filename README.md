# AI Sentiment Analyzer

This is a beginner NLP project built using Python, TextBlob and Streamlit.

The app analyzes text sentiment and predicts whether the sentiment is:
- Positive
- Negative
- Neutral

## Technologies Used
- Python
- NLP
- TextBlob
- Streamlit
